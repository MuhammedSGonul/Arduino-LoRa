#Arduino-LoRa E32 EByte Example

Pinouts:

| LoRa |  | Arduino |
|GND| <--> |GND|
|VCC| <--> |5V|
|AUX| <--> | |
|TXD| <--> |D3|
|RXD| <--> |D2|
|M1| <--> |D4|
|M0| <--> |D5|

| LoRa | Center-aligned | Right-aligned |
| :---         |     :---:      |          ---: |
| git status   | git status     | git status    |
| git diff     | git diff       | git diff      |
